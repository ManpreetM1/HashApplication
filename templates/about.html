About


This is a hashing application, which allows you to enter a message or upload a file and use a specific cryptographic hashing algorithm to generate a hash. This hash can then be used for many different kinds of applications, like quickly verifying a file or message matches another instance of that same file or message.

This application was created by me to provide a quicker and smoother way to calculate a hash compared to some of the solutions that exist online when you search them up. A lot of them don't use AJAX for a more responsive interface, or they are littered with ads and it can slow down the experience a lot. Using AJAX to update the frontend and flask in the backend to calculate the hashes allows for this app to be a much smoother and more responsive experience.

To access the file uploading you can simply open the side menu using the icon in the top left and click on the upload file anchor tag. If you simply want to create a hash of a message you can type or copy-paste it into the text area on the home page. In either case, there is a dropdown menu where you can choose from the currently supported hashing algorithms.

You can find a link to the github repo here: https://github.com/ManpreetM1/HashApplication
